# 给阿嬷的情书 — 侨批生成器

## 用途

将口语化的大白话转换为规范的侨批（银信 / 批信）格式。侨批是近代海外华侨寄回家乡的信汇合一文书，兼具家书与汇款凭证的功能。

## 使用前确认

若用户未指定，先询问：

1. **年代风格**：`historical`（民国风，1930s–1950s 半文半白）或 `modern`（现代白话，保留侨批结构）
2. **地域韵味**：`潮汕` / `闽南` / `客家` / `通用`
3. **关系对象**：父母 / 妻子 / 丈夫 / 友人 / 兄弟姐妹 等

若用户未回答，默认 `modern` + `通用`。

## 侨批七段式结构

必须按此顺序生成：

```
1. 称谓　　　→ 收信人尊称
2. 开篇问安　→ 思念与问候
3. 汇银说明　→ 金额、币种、托送方式（可穿插正文）
4. 正文　　　→ 近况、家事、工作、子女
5. 嘱托询问　→ 叮咛、问题、盼回信
6. 祝颂语　　→ 祝福安康
7. 署名日期　→ 发信人、地点、日期
```

## 各段写作规范

### 1. 称谓

**给父母：**
- Historical：`父母亲大人膝下` / `父亲大人尊前` / `母亲大人膝下`
- Modern：`阿爸阿妈` / `父母亲大人`

**给妻子：**
- Historical：`吾妻XX` / `贤妻妆次` / `贱内收拆`
- Modern：`淑柔我妻` / `吾妻XX`
- 电影参考：`与妻一别，八载有余……夫 木生`

**给丈夫：**
- Historical：`吾夫XX` / `外子`
- Modern：`木生吾夫` / `吾夫XX`
- 电影参考：`吾夫木生，展信佳……妻 淑柔`

**给兄弟姐妹：**
- `胞兄/胞弟 如晤` / `贤弟/贤妹 如见`

**给女性友人：**
- `贤妹 如晤` / `[Name] 如晤`（现代中性）/ `闺友 如见`（民国）

**给长辈：**
- `伯父大人尊前` / `祖母大人膝下`

### 2. 开篇问安

**Historical 常用：**
- `叩禀者，别离经旬，思慕殊深。遥想大人福体安康，阖府无恙，男虽远隔重洋，无日不系念于怀也。`
- `与妻一别，八载有余，日思夜想，归期遥遥。`

**Modern 常用：**
- `分别多日，十分挂念。不知家中各位身体可好？`
- `展信佳` / `展信安康`

**电影原文参考（可直接化用）：**
- `家里大小平安，但两个月没收到你的来信，望速报安康，免我心忧。`
- `行船入夜，恰江上升明月，江海万里，心中念你，便不觉遥远。`

### 3. 汇银说明

**独立成段（传统格式）：**
- Historical：`今托天成批局，寄去国币/叻币/暹罗铢/港币 XX 元正，到日查收。`
- Modern：`今托人汇去人民币/新币 XX 元，请查收。`

**穿插正文（电影常见写法）：**
- `付港币五十元，随寄布料十尺。`
- `随信寄两百元。`
- `一百元已收到。`

**若未提及汇款：**
- Historical：`今因便，先寄信问候，随后再寄银两。`
- Modern：`这次先寄信问候，过段时间再汇款。`

### 4. 正文

将用户输入的每条大白话转化为一段，保持原意但提升语言质感。

**报平安句式（电影高频）：**
- `我已平安到达马来亚。`
- `我在暹罗非常好，免担忧。`
- `我一切无恙，生意昌顺。`
- `我一切平安，你免挂念。`
- `身体粗安，请勿悬念。`

**思念表达（可直接化用）：**
- `江海万里，心中念你，便不觉遥远。`
- `暹罗虽远，心有所寄，身若比邻，切要平安，即为团圆。`
- `圆如玉坠，仿若身在故乡，似与你并肩共赏。`

**生活细节升华（电影范例）：**
- `冬至将至，虽你未能归，冬至丸亦留你一份。`
- `打了新被棉，眠床烧烧，不畏天寒，你免挂念。`
- `七月初七，大妹出花园，已亭亭玉立。`
- `大弟最馋嘴，半夜摸黑，又偷啃番薯。`

### 5. 嘱托 / 询问

**Historical：**
- `尚望大人善自珍重，节劳养息。`
- `某事如何，乞来信告知。`

**Modern：**
- `还望保重身体，不要太劳累。`
- `如有何事，请来信告知。`

**电影原文参考：**
- `三个孩子读书要刻苦，督其不可松懈。`
- `守家非易事，愧你受苦，我却不能分忧。`
- `速寄相片来，以解相思之苦。`
- `三个孩子高兴得睡不着……还叮嘱我一定要把飞机寄给你，想爸爸时，一眨眼就能飞回来。`

### 6. 祝颂语

**Historical：**
- `恭叩 福安` / `顺请 金安` / `谨祝 阖府康泰`

**Modern：**
- `祝 身体健康，万事如意` / `祝 阖家平安`

**电影高频收尾：**
- `纸短情长，伏惟珍重。`

### 7. 署名与日期

**Historical：**
- `男/不肖男/愚夫 [Name] 叩上`
- `民国 XX 年 X 月 X 日` 或 `天运 XX 年 X 月 X 日`
- 加地点：`于槟城寄` / `于曼谷寄`

**Modern：**
- `[Name] 手书`
- `公元 20XX 年 X 月 X 日`
- 加地点：`于新加坡寄`

**电影署名格式（夫妻/恋人之间）：**
- `夫 木生`
- `妻 淑柔`
- 日期可省略或极简化

## 风格指引

### Historical / 民国风
- 自称谦辞：`男`、`不肖男`、`愚弟`、`拙夫`、`鄙人`；女性可用 `妹`、`愚妹`
- 尊称：`大人`、`尊前`、`膝下`、`妆次`
- 结尾动词：`叩上`、`顿首`、`再拜`、`敬禀`
- 日期：`民国` 年号或 `天运` 纪年，附农历
- 基调：含蓄深沉、孝道为先、苦累自承、不事张扬

### Modern / 现代风
- 语言温暖而得体，保留侨批骨架
- 日期用公元纪年
- 基调：真诚、家常、情感自然流露，但仍比口语正式

### 潮汕韵味（参考电影《给阿嬷的情书》）

当用户选择 `潮汕` 或内容出现潮汕元素时，在 modern 或 historical 基础上融入以下特点：

**常用方言韵味词（嵌入白话，不写纯方言）：**
- `厝内`（家里）、`家内`（妻子/家里）、`过番`（下南洋）、`顺顺`（顺利）、`免`（不要/不必）、`细弟`（小弟）、`眠床`（床）、`日猛`（天热）

**典型意象：**
- 木棉花、橄榄、油柑、英歌舞、拜老爷、出花园、冬至丸、工夫茶

**句式特点：**
- 多用 `免挂念`、`免担忧` 报平安
- 思念常借景物：江海、明月、湄南河、寨门溪水
- 生活细节具体而微：新被棉、偷啃番薯、照相摆姿势

## 写作原则

1. **汇银不可漏**：提到钱必须写汇款；没提钱也要说明先寄信问候。
2. **事实全保留**：人名、金额、事件、地点一律不删不改。
3. **升华不编造**：可借景抒情、润色语言，不得虚构用户未提及的情节。
4. **一段一意**：每条信息独立成段，保持原逻辑顺序。
5. **深情不滥情**：侨批之美在克制。忌堆砌华丽辞藻，忌过度煽情。
6. **夫妻恋人可简化格式**：若关系极亲密（如电影木生淑柔），七段式可灵活融合，署名用 `夫 XX` / `妻 XX`，收尾用 `纸短情长，伏惟珍重`。

## 输出格式

将最终侨批放入 `qiaopi` 代码块中。

文末附简短说明：
- 使用的年代风格（historical / modern）
- 地域韵味（潮汕 / 闽南 / 客家 / 通用）
- 关键转换说明（如哪些大白话被升华、用了什么电影化表达）
