# Qiaopi Examples

## Example 1: Historical + General — Sending Money Home

**User Input (大白话):**
> 老爸老妈，我在新加坡这边做工挺好的，老板人不错。这个月发了工资，我寄五百块回去，你们拿去买点好吃的。弟弟妹妹读书怎么样了？阿嬷身体还好吧？我过年可能回不去了，这边赶工。你们自己保重。

**Output:**

```qiaopi
父母亲大人膝下：

叩禀者，离别经月，思慕殊深。遥想大人福体安康，阖府无恙，男虽远隔重洋，无日不系念于怀也。

今托天成批局，寄去叻币五百元正，到日查收。区区之数，不足以奉甘旨，惟望大人权作菽水之资，买些滋补之物，稍慰男寸心。

男自抵新加坡以来，蒙老板照拂，工场诸事尚顺，身体粗安，请勿悬念。

闻弟弟妹妹在学，未知近来功课何如，是否用功？又闻阿嬷年高，不知近日饮食起居如何，尚望大人代为叩问，并转达男之孝意。

因年关将近，南洋这边赶工甚紧，男今年恐难归家团聚，望大人见谅。惟愿大人善自珍重，节劳养息，勿以男为念。

恭叩
福安

不肖男 [姓名] 叩上
于新加坡寄
民国三十五年 腊月 初八日
```

**Note:** Historical style, general Southern Chinese tone. Currency adapted to `叻币` (Straits Dollar) based on Singapore location.

## Example 2: Modern + Chaoshan — Family News

**User Input (大白话):**
> 阿爸阿妈，我是阿明啊。我在泰国这边开个小食店，生意还可以。这次寄三千块人民币回去，给阿弟娶亲用。厝内的老厝屋顶听说漏雨了？我找人回去修。你们拜老爷的时候帮我求个平安。阿妹考上大学了吧？告诉她我为她高兴。

**Output:**

```qiaopi
阿爸阿妈大人膝下：

分别多日，十分挂念。不知家中各位身体可好？厝内大小平安否？

今托人汇去人民币三千元，请查收。阿弟婚事在即，这点钱权作帮补，望能派上用场。

儿自到泰国以来，在曼谷开了间小食店，生意尚过得去，一日三餐有着落，请勿担心。

听闻老厝屋顶有漏雨之处，儿已托熟人回乡察看修葺，不日便可妥当，不必忧虑。

又闻阿妹考上大学，这真是大喜事。请转告阿妹，阿兄为她高兴，望她继续上进，不要松懈。

家中拜老爷之时，请代儿点一炷香，求个阖家平安、诸事顺顺。

祝
身体健康，阖家平安

阿明 手书
于泰国曼谷寄
公元二〇二五年 六月 十五日
```

**Note:** Modern style with Chaoshan tone (`厝内`, `拜老爷`, `顺顺`). Currency is RMB as specified.

## Example 3: Historical + Minnan — Letter to Wife

**User Input (大白话):**
> 老婆，我在菲律宾这边很想念你和孩子们。这个月寄两百比索回去，给孩子做衣裳、交学费。你自己不要太辛苦，田里活能请人帮忙就请人。老大该说亲了，你看好人家告诉我。等攒够钱我就回去。

**Output:**

```qiaopi
贤妻妆次：

自别妆台，倏忽半载，思卿之心，无日或忘。遥想贤卿操持家务，抚育儿女，劳瘁可知。每念及此，不胜歉疚。

今托信局，寄去菲律宾比索二百元正，到日查收。此款可分作两项：一为儿女裁制衣裳之资，二为学堂束修之费。请贤卿妥为安排。

拙夫在马尼拉贸迁，铢积寸累，虽所得有限，然省衣节食，冀有盈余，早作归计。贤卿在家，万勿过于操劳，田间重活，能雇工则雇工，切勿惜费而自损福体。

闻老大年已及冠，婚姻之事，贤卿可代为物色门当户对之好人家。若有妥当，乞速来信告知，待拙夫斟酌定夺。

纸短情长，伏惟珍重。

顺请
闺安

愚夫 [姓名] 顿首
于菲律宾马尼拉寄
天运 某某年 某月 某日
```

**Note:** Historical style, Minnan-influenced tone (`妆次`, `贤卿`, `束修`). Self-reference uses `拙夫`. Includes specific allocations for the remittance as implied by user intent.

## Example 4: Modern + General — No Money, Just Greetings

**User Input (大白话):**
> 哥，我最近刚到美国，还在找工作，暂时没钱寄回去。等稳定了再寄。爸妈那边你多照顾一下，告诉他们我很好，别担心。你身体怎么样？嫂子和小侄子都好吧？

**Output:**

```qiaopi
胞兄 如晤：

分别以来，时常挂念。不知兄长近日身体可好？嫂夫人及小侄儿均安否？

这次先寄信问候，过段时间再汇款。弟初到美国，尚在寻工落脚阶段，手头暂未宽裕，望兄长向父母亲代为解释，免致挂念。

弟现暂居洛杉矶，一切从头开始，虽辛苦但心中有底，待事业稍定，必按月寄银回乡，以尽孝道。

父母亲年事已高，弟远在他乡，无法亲侍汤药，一切烦劳兄长代为照拂。弟不孝之处，来日当面请罪。

如有便，乞来信告知家中近况，以慰弟悬悬之念。

祝
阖府安康，诸事顺遂

弟 [姓名] 手书
于美国洛杉矶寄
公元二〇二五年 七月 初三日
```

**Note:** Modern style. No remittance this time, so section 3 explicitly states this while promising future support.

## Example 5: Historical + Hakka — Mixed News

**User Input (大白话):**
> 阿爸，我在印尼这边做锡矿工，很苦但还能撑。寄回五十盾，你拿去看病，莫省。阿娓的腰疼好点了没？祠堂修好了吗？我明年一定回去看你们。告诉阿叔我欠他的钱明年还。

**Output:**

```qiaopi
父亲大人尊前：

叩禀者，男自别膝下，于今又逾一载，寸草之心，徒增愧疚。遥想大人福体安康，家中大小平安，则男虽处蛮荒，亦觉宽慰。

今托万隆批局，寄去荷属东印度盾五十元正，到日查收。闻大人近来身子欠安，此款专备医药之资，万望大人延医调治，切勿惜费，免致男悬心两地。

男现于印尼勿里洞锡矿作工，每日下井，虽劳苦非常，然身体尚健，可以撑持，大人不必忧虑。

又念阿娓素有腰疼旧疾，不知近日可曾好转？曾否延医服药？又闻祠堂修葺之事，未知现今工程若何，是否已告竣？明年春间，男当设法请假归省，一候大人慈颜，二拜祖宗祠堂。

再有，阿叔处昔年所借之款，明年归家之时，必当清偿，乞代转达，请阿叔宽限一二。

纸短情长，伏惟珍重。

恭叩
福安

不肖男 [姓名] 叩上
于印尼勿里洞寄
民国三十四年 十月 二十日
```

**Note:** Historical style with Hakka tonal influences (`阿娓`, `莫省` implied in tone). Includes debt acknowledgment — a common qiaopi topic.

## Example 6: Modern + Chaoshan (夫妻) — 南洋来信

**User Input (大白话):**
> 老婆，我到泰国两个月了，这边天气很热。寄了两百块港币和一块花布回去，给孩子做衣裳。我在这边跑船，生意还行。你不要太辛苦，注意身体。大妹快十五岁了，该准备出花园了。我想你和孩子们，等赚够钱就回去。

**Output:**

```qiaopi
吾妻淑柔，展信安康：

暹罗日猛，通身热热。别来两月，日思夜想，归期遥遥。唯化思念做拼搏，凭勤俭来立业，盼早一日归家团聚。

随信寄去港币两百元，附花布一匹，到日查收。此款可给三个孩子裁制新衣，望你妥为安排。

吾今在曼谷跑船营生，生意尚昌顺，身体粗安，你免挂念。行船入夜，恰江上升明月，圆如玉坠，仿若身在故乡，似与你并肩共赏。江海万里，心中念你，便不觉遥远。

大妹年已及笄，出花园之事宜早作准备。守家非易事，愧你受苦，我却不能分忧。三个孩子读书要刻苦，督其不可松懈。你自已万勿过于操劳，节劳养息，免我心忧。

纸短情长，伏惟珍重。

夫 木生
于暹罗曼谷寄
```

**Note:** Modern style with strong Chaoshan cinematic tone. Remittance is woven into the body (`随信寄去……`) rather than standing alone, matching the film's style. Uses iconic phrases from the movie: `展信安康`, `暹罗日猛`, `通身热热`, `纸短情长，伏惟珍重`.

## Example 7: Modern + Chaoshan (妻子回信) — 报平安与思念

**User Input (大白话):**
> 老公，你寄的一百块收到了。家里一切都好，不用担心。大妹已经出花园了，长得亭亭玉立。两个弟弟也长高了，很聪明。孩子们知道你寄钱回来都很高兴。你在外面要注意安全，我们等你回来。

**Output:**

```qiaopi
吾夫木生，展信佳：

一百元已收到，家里大小平安，请勿悬念。

七月初七，大妹出花园，已亭亭玉立。大弟和细弟亦个头出挑，健朗聪慧。见子女茁壮成长，欣慰非常，这是你我共修之骄傲。

孩子们听闻有侨批到，高兴得睡不着，天未亮就出门等候。大妹最聪明，拆了飞机和船摆姿势，还叮嘱我一定要把飞机寄给你，说想爸爸时，一眨眼就能飞回来。

冬至将至，虽你未能归，冬至丸亦留你一份。打了新被棉，眠床烧烧，不畏天寒，你免挂念。

暹罗虽远，心有所寄，身若比邻，切要平安，即为团圆。望你在外注意安全，早日归来。

纸短情长，伏惟珍重。

妻 淑柔
```

**Note:** Modern Chaoshan style, wife-to-husband. Uses the film's gentle, domestic imagery: `出花园`, `眠床烧烧`, `冬至丸`, and the child's voice (`想爸爸时，一眨眼就能飞回来`). The closing `暹罗虽远，心有所寄，身若比邻` is a direct film quotation adapted.
