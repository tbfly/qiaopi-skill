# letters-to-ama

## 用途

将口语化的大白话（大白话）转换为规范的侨批（银信 / 批信）格式。侨批是近代海外华侨寄回家乡的信汇合一文书，兼具家书与汇款凭证的功能。

## 触发条件

当用户请求包含以下关键词时触发：
- 写侨批、侨批格式、银信、批信、番批、唐山批
- 转换大白话为书信、生成传统家书
- letters-to-ama、给阿嬷的情书

## 执行流程

1. **确认偏好**（若用户未指定）：
   - 询问年代风格：`historical`（民国风，1930s–1950s 半文半白）或 `modern`（现代白话）
   - 询问地域韵味：`潮汕` / `闽南` / `客家` / `通用`
   - 询问关系对象：父母 / 妻子 / 丈夫 / 友人 / 兄弟姐妹 等
   - 若用户未回答，默认 `modern` + `通用`

2. **生成侨批**：按七段式结构生成书信
   - 称谓 → 开篇问安 → 汇银说明 → 正文 → 嘱托询问 → 祝颂语 → 署名日期

3. **输出结果**：将侨批放入 `qiaopi` 代码块，文末附风格说明

## 参数

| 参数 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| content | str | 用户输入的大白话内容 | 必填 |
| style | str | 年代风格：`historical` / `modern` | `modern` |
| region | str | 地域韵味：`潮汕` / `闽南` / `客家` / `通用` | `通用` |
| relation | str | 关系对象：父母 / 妻子 / 丈夫 / 友人 / 兄弟姐妹 | 自动推断 |

## 依赖

无外部依赖，纯文本生成 Skill。
